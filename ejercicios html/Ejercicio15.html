Tipos de selectores que entran en examen

Clase (class="")

Este es un ejemplo de un selector de clase. Aplica el color azul a este texto debido a la clase "mi-clase".

Un selector de clase se utiliza para aplicar estilos a elementos que comparten una misma clase. El selector de clase se usa con un punto (.) seguido del nombre de la clase.

Identificador (id="")

Este es un ejemplo de un selector de identificador. El texto de este párrafo está en negrita debido al id "mi-id".

El selector de identificador se utiliza para aplicar estilos a un único elemento que tiene un identificador único. El selector de identificador se usa con el símbolo de almohadilla (#) seguido del nombre del id.

Todos los elementos (*)

Este párrafo tiene la fuente configurada a Arial, ya que se aplica a todos los elementos en el documento.

El selector universal (*) se aplica a todos los elementos del documento. Es útil para aplicar un estilo común a toda la página.

Varios elementos (p, em, h1, h2)

Este párrafo y los otros elementos están coloreados de verde porque el estilo se aplica a varios tipos de elementos como p, em, h1, h2.

El selector de varios elementos se utiliza para aplicar el mismo estilo a múltiples tipos de elementos, separándolos con comas.

Elementos contenidos en otros elementos (ol li)

  1. Elemento 1 de la lista
  2. Elemento 2 de la lista
  3. Elemento 3 de la lista

Este estilo se aplica a los elementos li dentro de una lista ordenada (ol para aplicar un margen izquierdo.

El selector de elementos contenidos en otros elementos se utiliza para aplicar estilos a un tipo de elemento que está dentro de otro tipo de elemento.

Sobrevuelo (p:hover)

Este párrafo cambiará de color a rojo cuando pases el ratón sobre él.

El selector :hover se utiliza para aplicar un estilo cuando el usuario pasa el ratón sobre un elemento, como un párrafo, un enlace, o una imagen.
